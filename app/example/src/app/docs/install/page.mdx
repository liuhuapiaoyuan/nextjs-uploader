# 文档测试

### 安装

```
npm install nextjs-uploader
```

或者

```
yarn add nextjs-uploader
```

### 使用

```javascript
import { Uploader } from 'nextjs-uploader'

function MyComponent() {
  const handleUpload = (files) => {
    console.log(files)
  }

  return (
    <Uploader
      multiple
      accept='image/*'
      maxSize={1024 * 1024 * 5}
      onUpload={handleUpload}
    />
  )
}
```
